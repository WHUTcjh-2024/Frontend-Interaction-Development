核心计算公式

1. 毛细波表面张力核心关系(Kelvin公式)

$$\sigma = \frac{\rho \omega^2}{k^3}$$
$\rho$ 为液体密度，$\omega=2\pi f$ 为角频率，$f$ 为激励频率，$k$ 为毛细波波数

2. 衍射角度计算

$$\alpha = arctan\frac{H}{L}$$ $$\beta = arctan\left( \frac{\Delta x + H}{L} \right)$$ $$\delta = \beta - \alpha$$
$H = H_0 - h$ 为零级光斑相对水面高度，$L$ 为振动点到光屏水平距离

3. 波数计算

$$k = \frac{2\pi}{\lambda_0} sin\frac{\delta}{2} \left( sin\left( \alpha + \frac{\delta}{2} \right) + sin\left( \alpha - \frac{\delta}{2} \right) \right)$$
$\lambda_0 = 632.8\ \text{nm}$ 为氦氖激光器固定波长

4. 相对误差计算

$$E = \frac{|\sigma - \sigma_0|}{\sigma_0} \times 100\%$$
$\sigma$ 为实验测量值，$\sigma_0$ 为标准值
